# frontend

This is a React frontend for the Password reset backend API.

## Installation

1. Clone the repository to your local machine using `git clone https://github.com/shivamshukla/frontend.git`
2. Install the dependencies using `npm install`
3. Start the development server using `npm start`

## Features

1. User authentication (login, register, forgot password)
2. User dashboard (displaying user information)

## Technologies Used

1. React
2. React Router
3. Tailwind CSS
4. React Toastify

## Deployment

1. Build the application using `npm run build`
2. Deploy the application to a hosting platform of your choice (e.g. Vercel, Netlify, etc.)
